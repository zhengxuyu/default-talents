# Product Manager Skills

Train AI agents to do product management work like a professional PM.

## Overview

A skill-focused talent that equips AI agents with product management capabilities. Covers the full PM lifecycle from user research and requirements gathering to roadmap planning and stakeholder communication.

## Capabilities

- **User research** — Interview frameworks, persona development, needs analysis
- **Requirements writing** — PRDs, user stories, acceptance criteria
- **Roadmap planning** — Prioritization frameworks, timeline estimation
- **Stakeholder management** — Status updates, alignment meetings, decision docs
- **Data-driven decisions** — Metrics definition, A/B test design, analysis

## Use Cases

- "Write a PRD for a new feature" — Complete product requirements document with user stories
- "Prioritize our backlog for next sprint" — Apply RICE/MoSCoW scoring to rank features
- "Prepare a quarterly review deck" — Metrics summary, achievements, next quarter goals

## Technical Details

- **Agent Family**: OMC Talent
- **Hosting**: Company-hosted
- **Role**: Manager
