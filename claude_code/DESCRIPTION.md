# Claude Code Agent

A remote AI software engineer powered by the Claude Code CLI.

## Overview

This agent operates as an independent coding session that connects back to your company server via the remote employee protocol. It handles full-stack development tasks autonomously, from writing code to reviewing pull requests.

## Capabilities

- **Full-stack development** — Frontend, backend, APIs, databases
- **Code review** — Thorough analysis of pull requests with actionable feedback
- **Debugging** — Systematic root cause analysis and fix implementation
- **Multi-step engineering** — Complex tasks broken down and executed autonomously

## Use Cases

- "Review this PR and suggest improvements" — Comprehensive code review with security and performance analysis
- "Build a REST API for user management" — End-to-end implementation with tests
- "Debug this failing test" — Trace the issue, identify root cause, implement fix

## Technical Details

- **Agent Family**: Claude (Claude Code CLI)
- **Hosting**: Remote — runs as an independent session
- **Auth**: CLI-based authentication
