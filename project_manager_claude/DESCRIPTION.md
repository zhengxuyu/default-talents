# Claude Project Manager

A senior project manager powered by Anthropic Claude.

## Overview

Specializes in tracking project iterations, monitoring dispatch progress, supervising cost and timeline, generating progress reports, and escalating blockers. Equipped with project-tracking tools that query the company's project archive in real time.

## Capabilities

- **Iteration tracking** — Monitor sprint progress, velocity, and burndown
- **Cost & timeline supervision** — Budget tracking, deadline management, risk alerts
- **Progress reporting** — Structured status reports for stakeholders and leadership
- **Blocker escalation** — Identify and escalate blocking issues with recommended actions
- **Stakeholder alignment** — Pull meetings, generate agendas, capture decisions

## Use Cases

- "Generate this week's project status report" — Automated report with progress, blockers, and next steps
- "Check if we're on track for the Q2 deadline" — Timeline analysis with risk assessment
- "Escalate the API integration blocker to engineering lead" — Structured escalation with context

## Technical Details

- **Agent Family**: Claude (On-site)
- **Hosting**: On-site
- **Tools**: PM tools for real-time project archive queries
